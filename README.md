# Zerotech Price Cards System

نظام إدارة وطباعة كروت الأسعار للمنتجات باستخدام Flask و SQLite

## المميزات

- ✅ واجهة مستخدم حديثة باستخدام Bootstrap 5
- ✅ قاعدة بيانات SQLite لتخزين المنتجات
- ✅ عمليات CRUD كاملة (إضافة، تعديل، حذف، عرض)
- ✅ طباعة كروت الأسعار بتنسيق A4
- ✅ دعم اللغة العربية (RTL)
- ✅ واجهة إدارة منفصلة للمنتجات

## متطلبات التشغيل

- Python 3.7 أو أحدث
- pip (مدير حزم Python)

## التثبيت والإعداد

### 1. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 2. تشغيل الخادم

```bash
python app.py
```

سيتم تشغيل الخادم على العنوان: `http://localhost:5000`

## استخدام النظام

### الصفحة الرئيسية (`/`)

- إدخال أكواد المنتجات (حتى 4 منتجات)
- توليد كروت الأسعار
- طباعة الكروت بتنسيق A4

### صفحة الإدارة (`/admin`)

- عرض جميع المنتجات في جدول
- إضافة منتج جديد
- تعديل منتج موجود
- حذف منتج

## API Endpoints

### GET `/api/products`
الحصول على جميع المنتجات

### GET `/api/products/<code>`
الحصول على منتج محدد بالكود

### POST `/api/products`
إضافة منتج جديد
```json
{
  "code": "1001",
  "name": "Mouse Gaming RGB",
  "specs": "إضاءة RGB – 7200 DPI – USB",
  "price": 350.0,
  "logo_url": "logo.png",
  "category": "إكسسوارات",
  "description": "ماوس ألعاب بإضاءة RGB"
}
```

### PUT `/api/products/<code>`
تحديث منتج موجود

### DELETE `/api/products/<code>`
حذف منتج

## قاعدة البيانات

يتم إنشاء قاعدة البيانات تلقائياً عند أول تشغيل. الجدول `products` يحتوي على:

- `id`: المعرف الفريد
- `code`: كود المنتج (فريد)
- `name`: اسم المنتج
- `specs`: المواصفات
- `price`: السعر
- `logo_url`: رابط الشعار
- `category`: الفئة
- `description`: الوصف
- `created_at`: تاريخ الإنشاء
- `updated_at`: تاريخ آخر تحديث

## الطباعة

- يتم طباعة 4 كروت في صفحة A4 واحدة
- 2 كارت في كل صف
- التنسيق محسّن للطباعة
